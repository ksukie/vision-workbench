# 模块运行错误排查

[索引](./问题排查指南.md) | [English](../en/module-runtime-errors.md)

本页覆盖基础 CV 效果、全景重构、YOLO 检测、YOLO 分割和图像分类预测中的运行期错误。

## 基础 CV 处理失败

先 Reset，再重新应用效果。如果新增了自定义操作，确认它返回的是合法的灰度或彩色 NumPy 图像。

## 不支持的效果

使用 GUI 下拉框或 `service.list_effects()` 中已经注册的效果。自定义效果必须先注册再使用。

## 全景重构失败

常见原因：

- 手动点对少于 3 组。
- 点对分布过于集中。
- SIFT 找不到足够特征。
- 单应矩阵或仿射变换估计失败。

建议使用重叠区域清晰的左右图，把手动点分散放在共同区域，不要全部放在角落。

## YOLO 检测运行失败

确认模型已加载、摄像头已打开、image size 为正数。如果只在 CUDA 上失败，先改用 `Device=cpu`，区分模型/数据问题和 GPU 运行环境问题。

## YOLO 分割失败

确认任务类型和模型类型匹配，模型文件存在，image size 是合法整数。实时分割失败时，先用静态图片 Run Once 验证。

## 图像分类预测失败

确认图片可以正常打开，预训练权重或 checkpoint 文件存在，所选 device 可用。
