# 问题排查指南

[返回项目 README](../../../README.zh-CN.md) | [English](../en/README.md)

当安装命令、GUI 弹窗、摄像头流程、模型加载、数据集校验或训练运行失败时，从这里开始排查。Vision Workbench 的用户可见错误会尽量附带对应的排查文档路径。

## 按症状选择

| 看到的问题 | 优先查看 |
| --- | --- |
| 异常退出后仍存在 GUI 进程、摄像头占用、训练任务、内存或 GPU 资源残留 | [应急清理](./应急清理.md) |
| `vision-workbench` 找不到、Python 导入失败、PySide6 启动、路径或编码问题 | [环境问题](./环境问题.md) |
| Torch、torchvision、CUDA、`nvidia-smi`、`doctor` 检查问题 | [深度学习依赖](./深度学习依赖.md) |
| `.pt`/`.pth` 缺失、模型下载失败、下载不完整、checkpoint 错误 | [模型与权重](./模型与权重.md) |
| 图片打开/保存、JSON 点对、checkpoint、输出文件失败 | [数据与文件](./数据与文件.md) |
| 摄像头扫描、打开、读帧、截图、录屏、权限或占用问题 | [摄像头与视频](./摄像头与视频.md) |
| 分类数据集、YOLO `data.yaml`、标签、mask、训练中止 | [数据集与训练](./数据集与训练.md) |
| 基础 CV、全景重构、检测/分割运行期错误 | [模块运行错误](./模块运行错误.md) |
| 版本不一致、`pip show` 仍是旧值、检查更新、一键更新、wheel 构建或发布资产 | [打包与发布](./打包与发布.md) |

## 快速检查

除非具体文档另有说明，以下命令都在仓库根目录执行：

```bash
python --version
python -m pip --version
python -m pytest
python scripts/install_dependencies.py doctor
```

反馈问题时，请尽量提供完整弹窗或终端输出、执行过的命令、错误提示附带的文档路径，以及当前安装的是基础依赖、图像分类依赖还是 YOLO26 依赖。
