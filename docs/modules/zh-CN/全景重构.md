## 概览

全景重构模块支持左图（参考）与右图（待拼接）拼接、SIFT 自动匹配、人工控制点和辅助人工点选流程。用户界面以统一 Vision Workbench 桌面里的原生 Qt 页面为准。

## 启动

```bash
vision-workbench
```

在左侧导航打开 **全景拼接**，加载左图（参考）和右图（待拼接），选择自动或手动模式，然后重构并保存输出。算法会把右图变换到左图参考坐标系；如果示例图看起来左右相反，以界面上的“参考/待拼接”说明为准。

## Python API

```python
from panorama_reconstruction.api import reconstruct_panorama_from_paths

result = reconstruct_panorama_from_paths("left_reference.png", "right_to_stitch.png")
```

## 源码结构

```text
src/panorama_reconstruction/api/             对外 API
src/panorama_reconstruction/application/     重构工作流
src/panorama_reconstruction/configuration/   默认配置
src/panorama_reconstruction/domain/          数据模型
src/panorama_reconstruction/infrastructure/  图像 I/O 适配
src/panorama_reconstruction/processing/      SIFT 与手动重构算法
src/vision_workbench/desktop/                统一 Qt 界面
src/panorama_reconstruction/window/          旧 Tkinter 兼容/参考代码
```

## 二次开发

新增拼接算法、点选策略、融合或裁剪逻辑时，先修改 processing/application 层，再在 Qt 页面暴露控件。详见 [二次开发指南 - 全景重构扩展](../../二次开发指南.md#全景重构扩展)。
