## 概览

相机诊断模块用于发现本地摄像头、探测采集模式、打开实时预览、显示 FPS，并支持截图和录屏。用户界面以统一 Vision Workbench 桌面里的原生 Qt 页面为准。

## 启动

```bash
vision-workbench
```

在左侧导航打开 **相机诊断**，刷新摄像头，选择设备和配置，然后打开相机并保存输出。

## Python API

```python
from camera_diagnostics.api import discover_cameras

cameras = discover_cameras()
```

## 源码结构

```text
src/camera_diagnostics/api/             对外 API
src/camera_diagnostics/application/     相机工作流
src/camera_diagnostics/configuration/   默认配置
src/camera_diagnostics/domain/          数据模型
src/camera_diagnostics/infrastructure/  OpenCV 与平台适配
src/vision_workbench/desktop/           统一 Qt 界面
src/camera_diagnostics/window/          旧 Tkinter 兼容/参考代码
```

## 二次开发

新增摄像头后端、配置探测、截图、录屏或资源释放策略时，先修改 service/infrastructure 层，再在 Qt 页面暴露控件。详见 [二次开发指南 - 相机诊断扩展](../../二次开发指南.md#相机诊断扩展)。
