# 图像分类 README

[返回 README](../../../README.md) | [English](../en/image_classification_README.md) | [二次开发指南](../../二次开发指南.md#图像分类扩展)

## 概览

图像分类模块提供 ResNet18、MobileNetV3 Small 预测、预训练权重管理、数据集校验和基础训练。用户界面以统一 Vision Workbench 桌面里的原生 Qt 页面为准。

## 安装

按需安装图像分类依赖：

```bash
python scripts/install_dependencies.py classification
```

如果手动使用 `requirements-classification.txt` 安装依赖，建议再执行 `python scripts/install_dependencies.py doctor` 检查 Torch 构建。

## 启动

```bash
vision-workbench
```

在左侧导航打开 **图像分类**。页面分为“预测”和“训练”两个标签页。预测区支持预训练权重和自定义 checkpoint；训练区支持包含 `train/` 和 `val/` 类别文件夹的数据集。

新手可直接点击“创建示例数据”生成两类小数据集，再执行“检查训练环境”“应用推荐批量”“验证数据集”和“开始训练”。训练在独立进程中运行，界面显示每轮损失、验证准确率和最佳准确率，并提供“停止训练”。

## 命令行训练

```bash
image-classification-train --dataset path/to/dataset --model resnet18 --epochs 5
```

## Python API

```python
from image_classification.api import predict_with_pretrained

result = predict_with_pretrained("resnet18", "image.jpg", topk=5)
```

## 源码结构

```text
src/image_classification/api/             对外 API
src/image_classification/application/     预测与训练工作流
src/image_classification/configuration/   默认参数和路径
src/image_classification/domain/          数据模型
src/image_classification/infrastructure/  数据集、模型和权重适配
src/image_classification/processing/      TorchVision 后端
src/image_classification/runner.py        命令行训练入口
src/vision_workbench/desktop/             统一 Qt 界面
src/image_classification/window/          旧 Tkinter 兼容/参考代码
```
