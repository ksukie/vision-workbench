## 文档范围 / Documentation Scope

`docs/` 只保存需要随项目共享和维护的正式文档。维护者个人的调试记录、临时分析和本地开发笔记不属于正式项目文档，也不作为用户操作依据。

The `docs/` directory contains documentation that is maintained and shared with the project. Personal debugging records, temporary analysis, and local development notes are outside the official documentation set and are not user instructions.
